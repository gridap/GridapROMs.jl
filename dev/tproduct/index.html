ROManifolds.TProduct.AbstractRankTensor — Type
abstract type AbstractRankTensor{D,K} end

Type representing a tensor a of dimension D and rank K, i.e. assuming the form

a = ∑_{k=1}^{K} a_1^k ⊗ ⋯ ⊗ a_D^k
